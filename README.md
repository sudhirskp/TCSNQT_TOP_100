# TCS NQT TOP 100

This repository contains solutions to the top 100 coding questions for TCS NQT (National Qualifier Test) preparation.

## About

TCS NQT is a national-level test conducted by Tata Consultancy Services for recruiting fresh talent. This repository aims to help students prepare for the coding section by providing solutions to frequently asked questions.

## Resources

- [TCS NQT Coding Sheet - TakeUforward](https://takeuforward.org/interviews/tcs-nqt-coding-sheet-tcs-coding-questions#point_1)

## Repository Structure

The repository is organized with Java solutions for various coding problems commonly asked in TCS NQT interviews.

## How to Use

1. Clone the repository
2. Navigate through the solutions
3. Practice and understand the logic behind each solution
4. Refer to the TakeUforward coding sheet for comprehensive problem coverage
